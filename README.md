# 🍽️ Swaad - Online Food Delivery System

Swaad is a Java Full Stack Food Delivery web application inspired by Swiggy and Zomato. Users can browse restaurants, explore menus, add items to the cart, place orders, and manage their profiles. An Admin Panel is also included to manage restaurants, menus, users, and orders.

---

## 🚀 Features

### User Module
- User Registration & Login
- Browse Restaurants
- View Restaurant Menu
- Add to Cart
- Update Cart Quantity
- Remove Items
- Checkout
- Place Order
- View Profile
- Edit Profile

### Admin Module
- Admin Login
- Dashboard
- User Management
- Restaurant Management
- Menu Management
- Order Management
- Delivery Management

## ⚙️ How to Run

1. Clone the repository

```
git clone https://github.com/Rithika-JK/Swaad-Food-Delivery.git
```

2. Import into Eclipse

3. Configure Apache Tomcat

4. Import the MySQL database

5. Update database credentials in `DBConnection.java`

6. Run on Server
